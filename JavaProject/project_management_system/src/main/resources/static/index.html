项目介绍

用途：用来对个人项目进行管理,汇总管理项目中的数据,提供高视化项目管理

后端：采用 SpringBoot 开发,使用 MD5 进行密码加密验证,并且使用 RabbitMQ 和 Redis 作为缓存,token进行免密登录

前端：使用BootStrapV3 和 jQuery 进行开发

安全：对用户密码进行加密存储，防止数据泄露；对敏感操作进行身份验证和授权控制；防范常见的网络攻击，如SQL注入

性能优化：采用缓存策略，如Redis缓存数据
